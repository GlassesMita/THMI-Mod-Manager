# Informazioni su THMI Mod Manager

THMI Mod Manager è uno strumento sviluppato dalla comunità progettato per migliorare l'esperienza di gioco di Touhou Mystia Izakaya attraverso funzionalità complete di gestione delle mod.

## Informazioni sulla Versione

- **Nome dell'Applicazione**: THMI Mod Manager
- **Versione**: 1.0.0
- **Costruito con**: ASP.NET Core
- **Licenza**: Open Source

## Sviluppo

Questa applicazione è stata creata dalla comunità di Touhou Mystia Izakaya per fornire ai giocatori un'interfaccia facile da usare per gestire le modifiche al gioco mantenendo sicurezza e compatibilità.

## Funzionalità di Sicurezza

- Rilevamento vulnerabilità CVE per la sicurezza del player Unity
- Avvisi di sicurezza automatici per componenti obsoleti
- Processi sicuri di installazione e gestione delle mod
