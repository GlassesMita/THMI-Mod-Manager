# THMI 模组管理器 - 主页

欢迎来到 **THMI 模组管理器**！这是一个专为《东方夜雀食堂》设计的模组管理工具。

## 🎮 游戏文件检查

为了确保模组管理器能够正常工作，我们需要验证游戏文件是否完整。系统会自动检查以下文件：

### 必需文件
- **Touhou Mystia Izakaya.exe** - 游戏主程序文件
- **UnityPlayer.dll** - Unity 引擎核心库文件

### 文件状态说明
- ✅ **已找到** - 文件存在且可访问
- ❌ **缺失** - 文件不存在或无法访问

## 🔧 开发模式

如果您是开发人员或正在进行测试工作，可以通过配置文件启用**开发模式**。在此模式下：
- 跳过游戏文件完整性检查
- 显示开发模式提示
- 允许在缺少游戏文件的情况下运行

## 📁 文件位置

请确保模组管理器与游戏文件位于同一目录下：
```
游戏目录/
├── Touhou Mystia Izakaya.exe
├── UnityPlayer.dll
└── THMI Mod Manager.exe
```

## 🌐 多语言支持

THMI 模组管理器支持多种语言，包括：
- 简体中文
- 繁体中文
- English (US)
- 日本語

## 🚀 开始使用

1. 确保游戏已正确安装
2. 将模组管理器放置在游戏目录
3. 运行 THMI Mod Manager
4. 开始管理您的模组！

## 💡 提示

- 如果显示文件缺失，请检查游戏是否正确安装
- 确保您拥有游戏的合法副本
- 建议在运行模组管理器前关闭游戏
