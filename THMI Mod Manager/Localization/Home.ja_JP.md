# THMI Mod Manager - ホーム

**THMI Mod Manager**へようこそ！これは東方夜雀食堂専用のMod管理ツールです。

## 🎮 ゲームファイル検証

Mod Managerが正しく動作するためには、ゲームファイルが完全であることを確認する必要があります。システムは以下のファイルを自動的にチェックします：

### 必要なファイル
- **Touhou Mystia Izakaya.exe** - メインゲーム実行ファイル
- **UnityPlayer.dll** - Unityエンジンコアライブラリファイル

### ファイルステータス表示
- ✅ **見つかりました** - ファイルが存在し、アクセス可能です
- ❌ **見つかりません** - ファイルが存在しないか、アクセスできません

## 🔧 開発モード

開発者である場合、またはテスト作業を行っている場合、設定ファイルで**開発モード**を有効にできます。このモードでは：
- ゲームファイル整合性チェックをスキップ
- 開発モード通知を表示
- ゲームファイルが存在しない状態での動作を許可

## 📁 ファイルの場所

Mod Managerがゲームファイルと同じディレクトリにあることを確認してください：
```
ゲームディレクトリ/
├── Touhou Mystia Izakaya.exe
├── UnityPlayer.dll
└── THMI Mod Manager.exe
```

## 🌐 多言語サポート

THMI Mod Managerは複数の言語をサポートしています：
- 簡体中文
- 繁體中文
- English (US)
- 日本語

## 🚀 始め方

1. ゲームが正しくインストールされていることを確認
2. Mod Managerをゲームディレクトリに配置
3. THMI Mod Managerを実行
4. Modの管理を開始！

## 💡 ヒント

- ファイルが見つからないと表示される場合は、ゲームが正しくインストールされているか確認してください
- ゲームの正当なコピーを所有していることを確認してください
- Mod Managerを実行する前にゲームを閉じることをお勧めします
