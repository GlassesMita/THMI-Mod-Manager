# About THMI Mod Manager

THMI Mod Manager is a community-developed tool designed to enhance the Touhou Mystia Izakaya gaming experience through comprehensive mod management capabilities.

## Version Information

- **Application Name**: THMI Mod Manager
- **Version**: 1.0.0
- **Built with**: ASP.NET Core
- **License**: Open Source

## Development

This application was created by the Touhou Mystia Izakaya community to provide players with an easy-to-use interface for managing game modifications while maintaining security and compatibility.

## Security Features

- CVE vulnerability detection for Unity player security
- Automatic security warnings for outdated components
- Safe mod installation and management processes
