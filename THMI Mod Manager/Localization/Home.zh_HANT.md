# THMI 模組管理器 - 首頁

歡迎來到 **THMI 模組管理器**！這是一個專為《東方夜雀食堂》設計的模組管理工具。

## 🎮 遊戲檔案檢查

為了確保模組管理器能夠正常運作，我們需要驗證遊戲檔案是否完整。系統會自動檢查以下檔案：

### 必需檔案
- **Touhou Mystia Izakaya.exe** - 遊戲主程式檔案
- **UnityPlayer.dll** - Unity 引擎核心庫檔案

### 檔案狀態說明
- ✅ **已找到** - 檔案存在且可存取
- ❌ **缺失** - 檔案不存在或無法存取

## 🔧 開發模式

如果您是開發人員或正在進行測試工作，可以透過設定檔啟用**開發模式**。在此模式下：
- 跳過遊戲檔案完整性檢查
- 顯示開發模式提示
- 允許在缺少遊戲檔案的情況下運行

## 📁 檔案位置

請確保模組管理器與遊戲檔案位於同一目錄下：
```
遊戲目錄/
├── Touhou Mystia Izakaya.exe
├── UnityPlayer.dll
└── THMI Mod Manager.exe
```

## 🌐 多語言支援

THMI 模組管理器支援多種語言，包括：
- 簡體中文
- 繁體中文
- English (US)
- 日本語

## 🚀 開始使用

1. 確保遊戲已正確安裝
2. 將模組管理器放置在遊戲目錄
3. 執行 THMI Mod Manager
4. 開始管理您的模組！

## 💡 提示

- 如果顯示檔案缺失，請檢查遊戲是否正確安裝
- 確保您擁有遊戲的合法副本
- 建議在運行模組管理器前關閉遊戲
