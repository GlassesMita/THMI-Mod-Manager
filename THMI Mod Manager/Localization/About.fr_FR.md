# <ruby title="You know too much">Tou<rt>T</rt>hou<rt>H</rt> Mystia<rt>M</rt> Izakaya<rt>I</rt></ruby> Mod Manager

THMI Mod Manager est une application permettant de gérer les mods de Touhou Mystia's Izakaya.

## Fonctionnalités

- **Gestion des mods** : Installez, mettez à jour et désinstallez facilement les mods de Touhou Mystia's Izakaya
- **Support multilingue** : Disponible en plusieurs langues, notamment le français, l'anglais, le japonais et le chinois
- **Interface conviviale** : Interface simple et intuitive conçue pour une facilité d'utilisation
- **Intégration Steam** : Intégration transparente avec Steam pour une gestion facile du jeu
