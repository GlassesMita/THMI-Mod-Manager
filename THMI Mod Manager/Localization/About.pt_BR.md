# Sobre o THMI Mod Manager

THMI Mod Manager é uma ferramenta desenvolvida pela comunidade projetada para aprimorar a experiência de jogo de Touhou Mystia Izakaya através de funcionalidades abrangentes de gerenciamento de mods.

## Informações da Versão

- **Nome da Aplicação**: THMI Mod Manager
- **Versão**: 1.0.0
- **Construído com**: ASP.NET Core
- **Licença**: Código Aberto

## Desenvolvimento

Esta aplicação foi criada pela comunidade de Touhou Mystia Izakaya para fornecer aos jogadores uma interface fácil de usar para gerenciar modificações do jogo mantendo segurança e compatibilidade.

## Recursos de Segurança

- Detecção de vulnerabilidades CVE para segurança do player Unity
- Avisos de segurança automáticos para componentes desatualizados
- Processos seguros de instalação e gerenciamento de mods
