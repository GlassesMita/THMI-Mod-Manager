# <ruby title="Ты слишком много знаешь">Tou<rt>T</rt>hou<rt>H</rt> Mystia<rt>M</rt> Izakaya<rt>I</rt></ruby> Mod Manager

THMI Mod Manager — это приложение для управления модами Touhou Mystia's Izakaya.

## Функции

- **Управление модами**: Легко устанавливайте, обновляйте и удаляйте моды для Touhou Mystia's Izakaya
- **Поддержка нескольких языков**: Доступно на нескольких языках, включая русский, английский, японский и китайский
- **Удобный интерфейс**: Простой и интуитивно понятный интерфейс, разработанный для удобства использования
- **Интеграция со Steam**: Бесшовная интеграция со Steam для удобного управления игрой
