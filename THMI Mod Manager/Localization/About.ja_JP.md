# <ruby title="You know too much">Tou<rt>T</rt>hou<rt>H</rt> Mystia<rt>M</rt> Izakaya<rt>I</rt></ruby> Mod Manager について

THMI Mod Managerは東方ミスティアの居酒屋のModを管理するためのアプリケーションです。

## 機能

- **Mod管理**：東方ミスティアの居酒屋のModの簡単なインストール、更新、アンインストール
- **多言語対応**：英語、日本語、中国語など複数の言語で利用可能
- **使いやすいインターフェース**：使いやすさを考慮したシンプルで直感的なインターフェース
- **Steam連携**：Steam とのシームレスな連携による簡単なゲーム管理
